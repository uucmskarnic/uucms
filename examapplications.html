Important – Pre Activity:
1. Please Verify all the course code before paying the fees.
2. The fees will be based on fee set by university:
    I. Income limit
    II. Gender
    III. Category
    IV. Reservation claim status of Student
    V. Amount will be visible after university set the fee.
3. QP Codes need to be sets for all courses set in the curriculum.
4. The registered curriculum of the Student should be Approved by Principal and also admission should be approved by Registrar.
5. Application fee paying between start date of application pay and last date of application fee (2nd Penalty set in exam notification).
6. If pay button is missing in new application here might be the following reasons:
    I. Curriculum not set.
    II. Curriculum not got approved.
    III. QP Codes not set for all curriculum.
    IV. Exam fee was not set by university.
7. Course registration for Eligible notification is mandatory to Apply for exam.
8. Special case late fee has to be set in general fee and special case last date penalty in exam notification has to be set.
Important – Post Activity:
Hall Ticket will be generated based on exam application fee paid and got approved.
